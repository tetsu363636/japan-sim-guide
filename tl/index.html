Sinasagot ang bawat tanong ng mga dayuhan

Maaari bang mag-sign up nang walang Japanese address?

Kailangan ng Japanese address para makumpleto ang pagpaparehistro. Ang pansamantalang address (hotel, guesthouse) ay maaaring gamitin, ngunit kailangang i-update pagkatapos. Kailangan ng Residence Card (在留カード) o My Number Card para sa pagpapatunay ng pagkakakilanlan.

Kailangan ba ng Japanese bank account o credit card?

Para sa Rakuten Mobile: Hindi. Maaaring magbayad gamit ang foreign Visa/Mastercard debit o credit card. Ito ay isang malaking kalamangan para sa mga bagong dating — ang pagbubukas ng Japanese bank account ay maaaring tumagal ng ilang linggo.

Maaari bang panatilihin ang kasalukuyang Japanese number? (MNP)

Oo. Humingi ng MNP transfer code sa kasalukuyang provider (online o sa telepono), pagkatapos ay ilagay ito sa pagpaparehistro. Ang proseso ay tatagal ng hindi bababa sa isang araw ng trabaho.

Totoo ba ang referral bonus ng empleyado?

Ito ay isang opisyal na programa ng Rakuten Group. Ang bonus (hanggang ¥14,000 Rakuten Points para sa MNP transfer) ay ibinibigay sa maraming installment sa loob ng ilang buwan pagkatapos ng activation. Ako ay empleyado ng Rakuten at ang aking link ay nasa ibaba.

Mayroon bang cancellation fee?

Ang Rakuten Mobile ay walang cancellation fee at walang minimum na kontrata. Maaaring mag-cancel anumang oras.

Paano mag-sign up sa Rakuten Mobile

Ang buong proseso ay maaaring gawin online sa loob ng humigit-kumulang 15 minuto.

1

Ihanda ang mga dokumento

Residence Card (在留カード) o My Number Card. Para sa MNP, ihanda rin ang transfer code mula sa kasalukuyang provider.

2

I-click ang referral link sa ibaba

Ang pag-apply sa pamamagitan ng employee referral link ay magbibigay ng bonus points. Huwag mag-apply nang direkta sa pamamagitan ng Rakuten homepage.

3

Piliin ang uri ng SIM

eSIM: agad na ma-activate sa pamamagitan ng QR code scan. Physical SIM: maihatid sa loob ng 2–3 araw ng trabaho.

4

Pagpapatunay ng pagkakakilanlan

Kumuha ng larawan ng Residence Card gamit ang telepono at i-upload. Ang AI verification ay karaniwang natapos sa loob ng ilang minuto.

5

I-activate at i-install ang Rakuten Link

eSIM: i-scan ang QR code para ma-activate agad. I-download ang Rakuten Link app para sa libreng walang limitasyong domestic na tawag.
